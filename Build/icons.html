Lunacon Icons

Icons

The Icon set is part of the Lunacon Font package and is very easy to use as icon or unicode. As a font you simply need to create a <i></i> element and adding the following classes lunacon lunacon-{name} 

<i class="lunacon lunacon-{name}"></i>
<!-- example -->
<i class="lunacon lunacon-moon"></i>
<i class="lunacon lunacon-moon-solid"></i>
<i class="lunacon lunacon-bell"></i>

Icon Sizes

There are 5 predefined font sizes to choose from. Simply add lunacon--size-x{factor}. Factor range 1 - 5

      
<i class="lunacon lunacon-moon-solid lunacon--size-x1"></i>
<i class="lunacon lunacon-moon-solid lunacon--size-x2"></i>
<i class="lunacon lunacon-moon-solid lunacon--size-x3"></i>
<i class="lunacon lunacon-moon-solid lunacon--size-x4"></i>
<i class="lunacon lunacon-moon-solid lunacon--size-x5"></i>
