Start your next project faster with less setup

We have create a couple of way to start you next Luna project faster.

Laravel-Luna-preset

For laravel lovers out there where have created a Luna preset package that will setup your sass, javascript and laravel-mix configuration. This package will also override the view and auth files using so artisan commands.

Instruction

  1. Install Laravel project as normal.
  2. cd into project
  3. Run composer require designbycode/laravel-luna-presets
  4. Now run php artisan preset luna
  5. Then run yarn or npm install
Setup _settings.sass file
This will now have copied all the necessary files for you. All wat you need to do now is copy the the content of the _settings.sass file in node_modules/luna-sass/Framework/sass directory to your laravel/resouces/assets/sass/_settings.sass file. Or simply run php artisan preset luna again after npm packages has been installed. The command will copy the _settings.sass file if it exists in the node_modules.
Artisan commands
Artisan command will create the auth views
code php artisan luna:view and then npm run watch
